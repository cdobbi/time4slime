# Time4Slime Website

A fun and colorful webpage for Time4Slime, a slime business created by a little girl!

## About

This website showcases popular slime products available for kids in 2026. It includes sections for about the business, products with images and wiggle animations, and contact information.

## Files

- `index.html`: The main HTML structure with updated product listings
- `style.css`: CSS styles with added animations and image styling
- `script.js`: JavaScript for interactive elements

## Features

- Product grid with placeholder images
- Wiggle animation on product hover
- Click effects for extra fun
- Responsive design

## How to View

Simply open `index.html` in any web browser or run a local server (e.g., `python -m http.server`).

## Customization

- Replace placeholder images with real slime photos
- Update product details or add more items
- Modify animations in `style.css`
- Add your daughter's name in the about section

## Troubleshooting

- If images don't load, check internet connection (using online placeholders)
- For best viewing, use a modern web browser
- Animations require CSS support
